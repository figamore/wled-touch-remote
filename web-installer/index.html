Install WLED Touch Remote

Flash the firmware onto a capacitive ESP32 Cheap Yellow Display directly from the browser.

This browser doesn't support Web Serial. Open this page in Chrome or Edge on a desktop computer. Web Serial needs a secure connection. Use the HTTPS link, not a local file.

Checking latest release… · Ensure a capacitive CYD is connected via USB

The installer performs a clean flash for reliable setup. Saved settings on the remote may be reset.

  1. Connect

    Plug the CYD into your computer with a data-capable USB cable.

  2. Install

    Click Install, then choose the ESP32 serial port when your browser asks.

  3. Pair

    Go to Config → WiFi & Network, check "Enable ESP-NOW" then copy the MAC from the Info tab into WLED's Linked MACs field.
